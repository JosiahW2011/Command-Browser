Commands for your terminal:

`/search <query>` - Do a basic Google search.

`/tab new <url>` - Open a new tab with the secified URL.
`/tab new` - Like `/tab new <url>`, but instead the URL is set to https://www.google.com/
`/tab close <tab-index>` - Close tab with the specified tab index.
`/tab close all` - Remove all tabs in Chrome.
